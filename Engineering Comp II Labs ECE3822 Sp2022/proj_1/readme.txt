This project creates playing cards and culminates in a creating a simple card game

####################################################################################
Part 1: Creating a card

Complete the code in <<card.cpp>> to create a <<card>> class that allows you to make
instances of a single playing card. Test your code with

./proj_1 0

####################################################################################
Part 2: Creating a pile of cards

Complete the code in <<cards.cpp>> to create a <<cards>> class that allows you to
make a pile consisting of many cards. Test your code with

./proj_1 1

####################################################################################
Part 3: Create a simple card game

Write code in the <<testcase_2()>> function in <<testcases.cpp>> to implement the
simple game explained in the comments. Test your code with

./proj_1 2

####################################################################################
Notes:

[a] Do not edit the Makefile
[b] Submit your code with
    ./submit.py
[c] start early, ask questions, and have fun!
